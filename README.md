# 🔙 ESC Back Navigation

[![Chrome Web Store](https://img.shields.io/badge/Chrome-Extension-blue)](https://chrome.google.com/webstore)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Расширение для Chrome, которое позволяет быстро переходить назад по истории браузера с помощью клавиши **ESC**.

## 📖 Описание

**ESC Back Navigation** — это простое и удобное расширение для навигации в браузере. Просто нажмите клавишу `ESC` на любой странице, чтобы вернуться на предыдущую страницу. Идеально подходит для тех, кто ценит скорость и эффективность!

### ✨ Возможности

- ⚡ **Быстрая навигация** — переход назад одним нажатием `ESC`
- 🎯 **Умная работа** — не мешает при вводе текста в полях и формах
- 🎨 **Визуальный индикатор** — анимация при переходе назад
- ⚙️ **Гибкая настройка** — включение/выключение через popup
- 🚫 **Исключения** — автоматическое отключение на Google Docs, GitHub и других важных сайтах
- 💾 **Сохранение настроек** — состояние синхронизируется через Chrome Sync
